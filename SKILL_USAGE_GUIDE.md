# 财经热点监控 Skill - 使用说明

## 📊 概述

这是一个专业的财经热点监控 AI 技能，能够实时扫描国内外权威媒体和社交平台，收集过去24小时内的财经热点事件，包括国际重大事件、资本市场数据（美股、伦敦期货、A股、港股），并特别关注科技领域和存储芯片行业的最新动态。

**重要更新**：✨ 现在每次查询都会自动生成精美的网页报告！

## 📱 手机端查看报告

### 方法一：Netlify Deploy（推荐）

1. **部署报告到云端**
   - 访问 https://app.netlify.com/drop
   - 将 `reports` 文件夹拖拽进去
   - 获得永久 URL

2. **手机添加快捷方式**
   - 手机浏览器打开该 URL
   - 添加到主屏幕
   - 命名为"财经热点"

3. **每次查询**
   - 电脑：双击 `auto-update.bat` 生成最新报告
   - 手机：点击图标打开，自动显示最新

### 方法二：本地服务器

```bash
# 在电脑上启动服务器
cd c:/Users/37334/CodeBuddy/20260228181507/reports
python -m http.server 8080

# 手机在同一 WiFi 下访问
http://192.168.1.100:8080/finance-hotspots-2026-03-01.html
```

### 方法三：GitHub Pages

1. 在 GitHub 创建仓库
2. 上传报告文件
3. 启用 GitHub Pages
4. 获得：`https://你的用户名.github.io/仓库名/`

## ⚙️ 配置参数

| 参数名 | 说明 | 默认值 |
|--------|------|--------|
| 收集时间范围 | 设置热点事件收集的时间范围 | 24h |
| 更新频率 | 设置数据收集的更新频率 | hourly |
| 跟踪股票代码 | 自定义跟踪的股票代码列表 | [] |
| 跟踪行业名称 | 自定义跟踪的行业名称列表 | [] |
| 优先关注领域 | 设置优先关注的领域 | [存储芯片, 人工智能, 科技] |
| 报告生成时间 | 设置每日报告生成的时间 | 18:00 |
| 是否推送通知 | 发现重要事件时是否推送通知 | true |
| 重要性阈值 | 设置事件重要性的最低阈值 | 3 |

## 🔄 工作流程

### 自动查询流程

1. **触发查询** → 用户输入"今日财经"
2. **收集数据** → 自动搜索各大媒体和市场数据
3. **分析处理** → 去重、分类、排序
4. **生成报告** → 自动创建 HTML 网页报告
5. **保存文件** → 保存到 `reports/` 目录
6. **显示链接** → 返回报告访问地址

## 🔧 常见问题

### Q: 如何查看历史报告？
A: 报告按日期保存在 `reports/` 目录，文件名为 `finance-hotspots-YYYY-MM-DD.html`

### Q: 报告可以自定义吗？
A: 可以编辑 `templates/finance-report.html` 模板文件

### Q: 如何设置自动定时生成？
A: 使用任务计划程序设置定时任务，每天运行 `auto-update.bat`

### Q: 手机总是看到旧报告怎么办？
A: 确保电脑运行 `auto-update.bat` 并上传到云端，或检查缓存
