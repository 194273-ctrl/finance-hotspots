# 财经热点监控系统 Skill

## 📊 概述

这是一个专业的财经热点监控 AI 技能，能够实时扫描国内外权威媒体和社交平台，收集过去24小时内的财经热点事件，包括国际重大事件、资本市场数据（美股、伦敦期货、A股、港股），并特别关注科技领域和存储芯片行业的最新动态。

## 🎯 核心功能

### 1. 多源媒体监控
- **国内权威媒体**：新浪财经、东方财富、证券时报、第一财经、财新网
- **国际权威媒体**：Bloomberg、Reuters、Wall Street Journal、Financial Times、CNBC
- **社交平台**：Twitter/X、Reddit、微博、雪球

### 2. 资本市场数据收集
- 🇺🇸 **美股市场**：S&P 500、NASDAQ、DOW JONES
- 🇬🇧 **伦敦期货**：原油、黄金、白银、铜、天然气
- 🇨🇳 **A股市场**：上证指数、深证成指、创业板指
- 🇭🇰 **香港股市**：恒生指数、国企指数、红筹指数

### 3. 科技与芯片专项监控
- AI 芯片、半导体行业动态
- 存储芯片（DRAM、NAND Flash）价格追踪
- 主要厂商（三星、SK海力士、美光、长江存储等）动态
- 云计算、5G、物联网等前沿科技

### 4. 个性化跟踪
- 📈 自定义股票代码跟踪
- 🏭 自定义行业名称跟踪
- ⭐ 智能推荐相关股票和行业
- 🔄 自动扩展收集范围

### 5. 网页报告生成
- 美观的 HTML 报告
- 结构化展示市场概览
- 热点事件分级展示
- 个性化关注项突出显示
- 风险提示

## 🚀 工作流程

### 工作流 1：媒体热点收集
- 扫描国内外权威媒体
- 扫描社交平台热门话题
- 去重和分类整理
- **触发**：每天 9:00, 13:00, 17:00 或手动触发

### 工作流 2：资本市场数据收集
- 收集美股指数和个股数据
- 收集伦敦期货商品价格
- 收集 A 股指数和个股数据
- 收集港股指数和个股数据
- 数据汇总分析
- **触发**：每小时、市场交易时间每30分钟或手动触发

### 工作流 3：科技领域和存储芯片专项监控
- 搜索科技领域热点
- 搜索存储芯片价格信息
- 搜索相关上市公司动态
- 生成行业分析报告
- **触发**：每天 10:00, 14:00 或手动触发

### 工作流 4：个性化股票跟踪
- 读取用户配置的股票和行业
- 搜索跟踪股票的最新信息
- 搜索跟踪行业的最新动态
- 自动扩展收集范围
- **触发**：用户配置更新、每小时或手动触发

### 工作流 5：生成网页报告
- 收集所有数据源
- 分析和排序（按重要性、时间、相关性）
- 生成报告结构
- 生成 HTML 报告
- 更新历史记录
- **触发**：每日 18:00、所有数据收集完成后或手动触发

## ⚙️ 配置参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 收集时间范围 | string | 24h | 设置热点事件收集的时间范围 (1h/6h/12h/24h/48h/72h) |
| 更新频率 | string | hourly | 设置数据收集的更新频率 (realtime/15min/30min/hourly/2hours/daily) |
| 跟踪股票代码 | array | [] | 自定义跟踪的股票代码列表 |
| 跟踪行业名称 | array | [] | 自定义跟踪的行业名称列表 |
| 优先关注领域 | array | [存储芯片, 人工智能, 科技] | 设置优先关注的领域 |
| 报告生成时间 | string | 18:00 | 设置每日报告生成的时间 (HH:MM) |
| 是否推送通知 | boolean | true | 发现重要事件时是否推送通知 |
| 重要性阈值 | number | 3 | 设置事件重要性的最低阈值 (1-10) |

## 📊 报告示例

生成的 HTML 报告包含以下部分：

1. **市场概览** - 展示美股、伦敦期货、A股、港股的实时数据
2. **今日热点事件** - 按重要性分级展示的财经热点
3. **科技与芯片专题** - 专门展示科技领域和存储芯片的信息
4. **个性化关注** - 展示用户跟踪的股票和行业动态
5. **风险提示** - 提醒重要的市场风险

## 🎨 报告模板

报告模板使用 Handlebars 模板语法，支持以下变量：

- `{{date}}` - 报告日期
- `{{updateTime}}` - 更新时间
- `{{newsCount}}` - 新闻数量
- `{{watchlistCount}}` - 关注项数量
- `{{usMarket}}` - 美股市场数据
- `{{londonMarket}}` - 伦敦期货数据
- `{{chinaMarket}}` - A股市场数据
- `{{hkMarket}}` - 港股市场数据
- `{{hotTopics}}` - 热点事件列表
- `{{chipPrices}}` - 存储芯片价格
- `{{techNews}}` - 科技新闻
- `{{watchlist}}` - 个性化关注列表
- `{{risks}}` - 风险提示
- `{{generationTime}}` - 报告生成时间
